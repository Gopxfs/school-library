# School Library
In this project, I'm learning about OOP by creating a school library app. It is possible to: add new students/teachers, add new books and save records of who borrowed a given book.

## Built With:
- Ruby

## Getting Started:
To access and change the code on your computer (using GIT):
> Note: if you don't use git, you can download the ZIP of this repository by clicking on the "Code" button above the files list.
### Setup
- Open the repository [link](https://github.com/Gopxfs/school-library).
- Click on the "Code" button, above the files list.
- Select one of the three options to copy the URL: HTTPS, SSH key and GitHub CLI.
- Open the Terminal in the directory you want to copy the project.
- Type git clone, paste the URL you copied, and press enter.
### Usage
To edit this project:
- Open the repository you cloned with a code editor that you like.
- Edit the code as you want.
